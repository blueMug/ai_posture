# AI Posture

一个微信小程序原型：调用微信原生相机，并在拍摄预览画面上叠加 PNG 姿势轮廓图，用于景点拍照、穿搭拍照等娱乐出片场景。

## 功能

- 姿势选择引导页
- 按 PNG 模板分类展示姿势轮廓图
- 点击姿势后再打开相机
- 原生相机全屏预览
- 直接使用 12 张透明 PNG 图片作为姿势模板
- 相机内支持拖动、双指缩放、右下角单指缩放、按钮微调、6 个尺寸档位和重置姿势轮廓图
- 模板左右切换
- 前后摄像头切换
- 显示或隐藏姿势轮廓图
- 拍照后预览
- 保存照片到系统相册

## 使用方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择当前目录：`/Users/bytedance/Documents/person/project/ai_posture`。
4. AppID 可以先使用测试号或替换为你的正式小程序 AppID。
5. 相机能力建议用真机预览测试，开发者工具模拟器对相机能力支持有限。
6. 进入小程序后，先在首页选择姿势，再进入相机拍照。

## 当前边界

- 姿势轮廓图只在拍摄预览时显示，不会合成到最终照片里。
- 暂不做人像识别、姿势识别或实时纠偏提示。
- 姿势模板不再维护骨架、关节点、代码绘制线条或图片提取工具，只直接引用 PNG 图片。
- `tmp/` 仅用于临时素材处理，已从小程序打包配置中忽略。
